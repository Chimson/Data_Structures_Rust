NeoVim Notes

Start:
% nvim file

QuickRef:
  "Esc" back to Normal  
  "i" switches to Insert at left of cursor, "a" at right 
  "I" starts insert at beginning of the line, 
  "A" starts insert after moving the cursor to the end of the line
  "dd" deletes a line, d in Visual mode deletes selection
  "D" is "d$" delete to end of the line
  "x" deletes char, "2x" is two chars, most commands do this
  "yy" yanks a line
  "u" is undo "ctrl-r" is redo
  "o" add line insert below cursor, "O" above cursor 
  "ZZ" writes the file and exits, ":q" no write and quits
  "!" is the override command modifier
  "w" moves to start of next word,"W" consider whitespace 
  "b" moves to the start of the previous word, "B" consider's whitespace, I mapped to "shift-larrow"
  "e" moves to the end of the next word, "E" whitespace, same as "shift-rarrow" 
  "^" to first char of the line, "0" beginning of line 
  "$" end of the line, no insert
  "G" puts cursor at the end of the file, "gg" puts at the beginning of the file 
  ";" repeats a command forward, "," repeats backwards
  "%" moves to matching (), [], {}
  "ctrl-U" scrolls up half a screen
  "ctrl-D" scrolls down half a screen
  "/string" (then enter) searches for "string" in the file, use \ for escape chars
  "n" is next occurance of string in /string,"N" repeats in the opposite direction
  ":noh" turns off the search highlight
  "ma" marks the position as mark a, have 26 available marks
  "mA" marks global marks across files, "`a" goes to mark a (line and col)
  "`." moves to last change mark, "`"" moves to last on file 
  "``" goes back to the last mark
  "p" puts the deleted/yanked line back, "P" puts before cursor
  "y" is a yank (copy)
  "R" is Replace mode, overwrites backspace goes back to previous, like undo
  ":w" saves old file, if there are unsaved changes
  ":wall" saves all open files
  ":qall" closes all, prompts if unsaved
  ":edit file.txt" closes current file and opens file.txt to edit  
  ":saveas othername.file" will write current file to a copy
  ":file newname.file" change the name of the file without writing 
  ":pwd" is current working directory
  ":cd path/to/dir" changes dir of nvim instance
  ":Ex" is the file explorer, "i" cycles styles
  ":browse oldfiles", but the selection doesn't work right
  ":tabedit newfile.txt" adds a tab, can use mouse for new tabs, close them
    can use :tab prefix with other commands 
  ":tabedit ." tabopen on file browser
  ":terminal zsh" will open a window for an interactive zshell
    go to insert mode to add commands
    may not open in the vim session pwd on each session load
  ":mksession newsession.vim" will save you current session 
    sessions are filelists, windowlayout, etc.
    saves the vim file in the current working dir
  ":source newsession.vim" opens the saved session
  "% vim -S newsession.vim" loads the session saved in cur dir 
  "% nvim file.lang" opens the file
  "% nvim ."  opens vim in the current dir
  "% vim -R file" opens file in read only, -M is more strict
  ":vsplit otherfile.txt" does a vertical split
  ":leftabove vsplit newfile.txt" opens on the left
  ":close" closes the current window
  "ctrl-W" jumps between windows 
  ":!ls" runs shell command ls
  "v" is visual mode, highlights text, see op-motion effects
  "V" is Visual Line mode, highlights lines
  after selection, using : gives range '<,'> meaning from start mark
    to end mark, can add command after it 
  "%:s/from/to/c" asks to change all "from" to "to" 
  ":LspRestart" restarts all lsp servers

  "zt" scrolls puts it at the top, "zb" at the bottom

  up down arrows go through search history, can add partial word to filter

  "*" on cursor word automatically adds it to search
    "g*" matches if it contains cursor word
  "#" is the same in the other direction
    "g#" matches if it contains cursor word
  "/string\>" where \> only matches at the end of the word
    in this case string
  "/\<string" where "\<"only matches the beginning of the string
  "/\<word\>" searches the exact word
  ":set noincsearch" turns off displaying matches while typing
  ":edit ~/.config/nvim/init.vim" opens the vim settings file

  "'a" goes to mark a beginning of line

  "dw" deletes a word from cursor start, "d4w" deletes 4 words
    same for "de", "d$" and other similar commands
    this is operator motion
  "daw" is delete a word, no matter where at in word  
  "cis" is change inner sentence (text object)
  "cas" is change a sentence, includes white space
  "cw" is change word: deletes word and stays in insert mode
  "cc" changes the whole line "dd", "c$", other op-motions
  "X" is "dh" delete char left of cursor  
  "C" is "c$" change to end of line
  "s" is "cl" change one char
  "rx" inserts x under cursor, stays in Normal mode
  "." repeats the last op-motion, does not include undo, redo, :cmd
  "~" changes the case  
  ":help E37" opens help on the vim error message
  % vim file1.txt file2.txt opens sequence of files
  ":next" edits next opened file, prompts if you want changes
  ":next!" abandons changes on old file
  ":wnext" save changes and move to the next file
  ":args" displays current list of files
  ":previous" and wprevious, first, last on other files in the list
  ":set autowrite" (also noautowrite) sets always write current file

    vim.opt.expandtab = true -- convert tabs to spaces 
    vim.opt.tabstop = 2 -- how many spaces a real tab counts for -
    vim.opt.shiftwidth = 2 -- indentation size -
    vim.opt.softtabstop = 2 -- how many spaces <Tab> inserts in INSERT mode
